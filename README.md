# SSAFY Home - Spring Framework

📌 **プロジェクト紹介**  
SSAFY Homeは全国のアパート取引情報を提供するWebアプリケーションです。既存のServletベースプロジェクトをSpring Frameworkにマイグレーションして、より堅牢で拡張可能なアーキテクチャを実装しました。ユーザーに迅速で正確な住宅情報を提供し、会員管理、関心地域設定、アパート取引情報照会などの機能を提供します。

## 🚀 **主要機能**

### 🔑 **会員管理**
- **会員登録**: ユーザーID、パスワード、メール、名前入力
- **ログイン**: IDおよびパスワード認証、セッション管理
- **パスワード検索**: ユーザー情報再認証後、メールで新しいパスワード送信
- **会員情報管理**: 会員情報照会、修正、退会機能

### 🏢 **アパート取引情報照会**
- **地域別照会**: 市道 → 区郡 → 邑面洞選択後、アパート取引情報照会
- **詳細情報**: アパート名、取引金額、面積、取引日付など詳細情報提供
- **地図連動**: 統計庁の統計地図APIを利用した地図可視化

### 🏬 **商圏照会**
- 商圏情報照会：地図でアパートをクリックすると周辺商圏情報を照会
- 詳細情報：業者名、業種分類、住所、建物名など詳細情報提供

### 🏢 **関心地域管理**
- **関心地域設定**: 町単位で関心地域登録/削除
- **関心地域照会**: 設定した関心地域内のアパート取引情報照会

### 📝 **掲示板機能**
- **投稿管理**: 投稿作成、照会、修正、削除
- **投稿検索**: 投稿に対する検索および詳細検索機能提供
- **投稿予約**: 投稿作成後、時間を選択すると該当時間に投稿公開

### **🛡️ 認証認可機能**
- Spring SecurityとJWT Tokenを活用した認証認可実装
- 未登録ユーザーは会員登録、ログイン、パスワード検索、取引履歴検索など基本的な機能のみ使用可能
- 登録されたユーザーは会員情報照会および修正、会員退会、関心地域設定および修正、投稿照会などの機能使用可能
- 管理者は掲示板管理、会員管理、会員削除などの機能を使用可能

## 📢 **コミットメッセージ作成規則**

### 📌 **コミットメッセージ構造**
各コミットメッセージは以下の2つのパートで構成されます。

- **タグ:** コミットの性質を表すキーワード
- **内容:** コミットの主要内容を簡潔に記述

### 🔖 **Commit Type**

| タグ       | 説明                             |
| ---------- | -------------------------------- |
| **Feat**   | 新機能追加                  |
| **Fix**    | バグ修正                         |
| **Docs**   | 文書修正                         |
| **Style**  | コードフォーマット、セミコロン漏れなど      |
| **Refactor**| コードリファクタリング                    |
| **Test**   | テストコード追加およびリファクタリング        |
| **Chore**  | ビルド業務修正、パッケージマネージャー修正 |
| **Init**   | 初期プロジェクト設定                |
